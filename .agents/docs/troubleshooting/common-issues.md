# Common Issues

**Module not found**: `pnpm install` • verify `@` alias
**ESLint**: `pnpm lint` (missing PropTypes, import order)
**Hydration**: use `useEffect` for client-only (dates, random, browser APIs)
**Image not loading**: check `next.config.js` remote patterns (Cloudinary)
**Form fails**: verify EmailJS/reCAPTCHA in `.env.local`
**Slow build**: check ISR times • simplify GraphCMS queries
**CLS**: set width/height on images • use skeletons
